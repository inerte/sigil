# Sigil Benchmarks

This directory holds language-level benchmark families.

Current layout:

- `tokens/` — active token-efficiency and tokenizer-shape benchmarks for Sigil syntax
- `developer-experience/` — active workflow and tool-loop benchmarks for coding agents
- `llm-generation/` — reserved for future codegen/eval benchmarks

Today the active benchmark families are:

- `tokens/`
- `developer-experience/`

Start here:

- developer-experience methodology and commands: `language/benchmarks/developer-experience/README.md`
