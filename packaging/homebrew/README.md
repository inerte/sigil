# Homebrew Packaging

This directory keeps the canonical Homebrew formula snapshot for Sigil and the checksum input used to generate it.

- `Formula/sigil.rb` is generated by `projects/homebrewPackaging`
- `SHA256SUMS` is the current release checksum snapshot used for local verification
- the generated formula installs the bundled `language/` and `runtime/` payloads under `share/sigil/`
- the generated formula declares `node` as a runtime dependency for `sigil run`, `sigil test`, `sigil validate`, `sigil inspect world`, and `sigil debug ...`

## Local generation

```bash
pnpm sigil:package:homebrew
```

## CI verification

```bash
pnpm sigil:test:homebrew
```

## Tap publishing

The release workflow can push the generated formula to `inerte/homebrew-tap` when the `HOMEBREW_TAP_DEPLOY_KEY` secret is configured. The tap repo should contain `Formula/sigil.rb` and no generation logic.
