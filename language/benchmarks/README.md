# Sigil Benchmarks

This directory holds language-level benchmark families.

Current layout:

- `tokens/` — active token-efficiency and tokenizer-shape benchmarks for Sigil syntax
- `developer-experience/` — reserved for future workflow and tool-loop benchmarks
- `llm-generation/` — reserved for future codegen/eval benchmarks

Today the only active benchmark family is `tokens/`.

Start here:

- methodology and commands: `language/benchmarks/tokens/README.md`
- published token results: `language/benchmarks/tokens/RESULTS.md`
